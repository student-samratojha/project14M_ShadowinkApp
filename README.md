# 🦇 ShadowInk: Secure Story Authentication System

ShadowInk ek cinematic, Batman-themed story writing platform hai. Yahan writers apni secret narratives ko ek secure aur brooding dark universe mein likh aur protect kar sakte hain.

## ✨ Features

- **Cinematic UI:** Gotham-inspired dark theme jisme smooth lightning animations aur 3D hover effects hain.
- **Secure Vault:** JWT-based authentication system jo har story aur identity ko protect karta hai.
- **Interactive Experience:** Orbitron typography aur moving grid backgrounds ke saath immersive feel.
- **Dynamic Stories:** EJS templating ka use karke real-time story archiving.

## 🚀 Tech Stack

- **Backend:** Node.js, Express.js
- **Frontend:** EJS (Embedded JavaScript), Tailwind CSS (CDN)
- **Database:** MongoDB (via Mongoose)
- **Animations:** Custom CSS3 Keyframes & Bezier Easing

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/project13M.git
   cd project13M
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Setup Environment Variables:**
   Root directory mein ek `.env` file banayein aur ye details bharein:
   ```env
   PORT=3000
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_batman_secret_key
   ```

4. **Run the application:**
   ```bash
   npm start
   ```
